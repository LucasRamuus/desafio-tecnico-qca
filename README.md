# Desafio Técnico - QCA

Projeto desenvolvido como parte do desafio técnico para a QCA.

## Objetivo
Realizar a ingestão de dados a partir de arquivos PDF, persistir os dados em JSON
e gerar análises utilizando Python, Pydantic e Pandas.

## Tecnologias
- Python 3
- Pydantic
- Pandas
- pdfplumber

## Estrutura
- Ingestão de PDFs
- Validação de dados
- Armazenamento em JSON
- Analytics
